Basic arithmetics

What basic arithmetics?

Arithmetic operations serve as the cornerstone of mathematics, providing the framework upon which more complex mathematical concepts are built. Through the seamless manipulation of numbers, addition, subtraction, multiplication, and division empower us to navigate myriad real-world scenarios with numerical precision.

Consider addition, the simplest of these operations, akin to gathering ingredients for a recipe. Just as we combine flour, sugar, and eggs to create a cake, addition amalgamates numbers to yield a sum greater than its parts.

Subtraction, on the other hand, acts as the kitchen scale, allowing us to discern the surplus or deficit between quantities, much like weighing ingredients to ensure recipe accuracy.

Moving to multiplication, envision a farmer planting rows of seeds in a vast field. With each seed representing a number, multiplication represents the farmer's action of sowing, where repeated addition efficiently propagates numerical growth.

Division, conversely, mirrors the farmer's harvest, partitioning the field into equal portions, akin to distributing a finite quantity among individuals equitably.

These operations go beyond just doing math — they're part of our everyday lives in many ways. Whether we're planning a household budget or sharing a pizza among friends, arithmetic operations help us make many decisions and take actions. They form the basic framework of mathematics, helping us solve problems, be precise, and share things fairly in a world ruled by numbers.

Addition

Definition

Addition is one of the four basic arithmetic operations. It involves combining two or more numbers to get a total or sum. The symbol for addition is "+".

Rules

  1. Commutative Property: The order of the numbers doesn't matter. For example:

    3 + 5 = 8 and 5 + 3 = 8

    6 + 8 = 14 and 8 + 6 = 14

  2. Associative Property: When adding three or more numbers, the way in which the numbers are grouped doesn't change the sum. For example:

    (2 + 3) + 4 and 2 + (3 + 4).

    (1 + 2) + 3 and 1 + (2 + 3)

  3. Identity Property: Adding zero to any number does not change the number. For example:

    7 + 0 = 7.

    100 + 0 = 100.

  4. Adding Positive and Negative Numbers: When adding a positive number and a negative number, subtract the smaller absolute value from the larger absolute value and keep the sign of the number with the larger absolute value. For example:

    5. 7 + (-3) = 4

    -5 + 8 = 3

Subtraction

Definition:

Subtraction is one of the four basic arithmetic operations. It involves taking one number away from another to find the difference. The symbol for subtraction is "−".

Rules

  1. Order Matters: Unlike addition, the order in which you subtract numbers matters. For example:

    10 - 5 = 5 and 5 - 10 = -5

    7 - 3 = 4 and 3 - 7 = -4

  2. Subtracting Zero: Subtracting zero from any number leaves the number unchanged. For example:

    7 - 0 = 7

    3. 100 - 0 = 100

  3. Subtracting a Number from Itself: Subtracting any number from itself results in zero. For example:

    1000 - 1000 = 0

    33 - 33 = 0

  4. Subtracting Negative Numbers: Subtracting a negative number is the same as adding its positive counterpart. For example:

    10 - (-3) = 10 + 3

    17 - (-10) = 17 + 10

Multiplication

Definition:

Multiplication is one of the four basic arithmetic operations. It involves adding a number to itself a certain number of times. The symbol for multiplication is "×" or "*".

Rules

  1. Commutative Property: The order of the numbers being multiplied does not change the product. For example:

    3 x 4 (3 + 3 + 3 + 3) is equal to 4 x 3 (4 + 4 + 4)

    2 x 5 (2 + 2 + 2 + 2 + 2) is equal to 5 x 2 (5 + 5)

  2. Associative Property: When multiplying three or more numbers, the way in which the numbers are grouped does not change the product. For example:

    (2 x 3) x 4 is equal to 2 x (3 x 4)

    (6 x 9) x 7 is equal to 6 x (9 x 7)

  3. Distributive Property: Multiplying a number by a sum is the same as doing each multiplication separately. For example:

    2 x (3 + 4) = 2 x 3 + 2 x 4

    5 x (7 + 8) = 5 x 7 + 5 x 8

  4. Multiplying by One: Any number multiplied by one remains unchanged. For example:

    7 x 1 = 7

    9 x 1 = 9

  5. Multiplying by Zero: Any number multiplied by zero is zero. For example:

    100 x 0 = 0

    10 x 0 = 0

Division

Definition:

Division is one of the four basic arithmetic operations. It involves splitting a number into equal parts. The number being divided is called the dividend, and the number you divide by is called the divisor. The result is called the quotient. The symbol for division is "÷" or "/".

  1. Division by One: Any number divided by one remains unchanged. For example:

    2 ÷ 1 = 2

    100 ÷ 1 = 100

  2. Division by Itself: Any number divided by itself equals one. For example:

    8 ÷ 8 = 1

    73 ÷ 73 = 1

  3. Division by Zero: Division by zero is undefined. For example:

    10 ÷ 0 is not possible

    67 ÷ 0 is not possible

  4. Zero Divided by a Number: Zero divided by any non-zero number is zero. For example:

    0 ÷ 7 = 0

    0 ÷ 100 = 0

  5. Distributive Property: Division distributes over addition and subtraction within parentheses. For example:

    (a + b) ÷ c = a ÷ c + b ÷ c

1. You have $50 and want to buy fruits and vegetables. If the fruits cost $20 and vegetables cost $15, how much money will you have left after the purchase?

2. You order a pizza and cut it into 8 slices. If there are 5 people and you want to divide the slices equally, how many slices will each person get, and how many slices will be left over?

3. You are planning a road trip that covers 300 miles. If you plan to stop for a break every 60 miles, how many breaks will you take during the trip?

4. A class project requires 24 hours of work. If there are 4 students in the group and they share the work equally, how many hours will each student need to contribute?

5. You are planning a party and need to buy snacks. If one pack of snacks costs $7 and you have $35, how many packs can you buy, and will you have any money left?
