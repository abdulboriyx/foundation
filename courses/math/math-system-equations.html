System of linear equations

What is system of linear equations?

A system of linear equations is a collection of two or more linear equations involving the same set of variables. The goal is to find the values of the variables that satisfy all equations simultaneously, making them true.

Types of system equations

  1. Consistent and Independent Systems: A consistent and independent system of linear equations is a system in which there is exactly one unique solution. This occurs when the equations represent lines (in two variables) or planes (in three variables) that intersect at precisely one point.
    • 2x + 3y = 6
      y = 2
  2. Consistent and Dependent Systems: A consistent and dependent system of linear equations is a system where the equations represent the same line or plane. This means there are infinitely many solutions because every point on the line or plane satisfies both equations.
    • 2x + 4y = 8
      x + 2y = 4
  3. Incosistent systems: An inconsistent system of linear equations is a system where the equations represent lines or planes that do not intersect at any point. This means there is no solution to the system because there are no points that satisfy all the equations simultaneously.
    • x + y = 2
      x + y = 5

Solving system of linear equations

When solving systems of linear equations with two variables, the goal is to find the point (or points) where the equations intersect. We will learn these methods: substituion method and elimination (or addition) method

Substitution method

Substituion method: The substitution method involves solving one of the equations for one variable and then substituting that expression into the other equation. This reduces the system to a single equation with one unknown, which can be solved directly.
Example:

  • x + 2y = 8
    3x - y = 7
  1. Solve one of the equations for one variable: We'll solve the first equation for x:
    • x + 2y = 8 to x = 8 - 2y
  2. Substitute the expression for x into the second equation: Substitute x = 8 - 2y into the second equations:
    • 3(8 - 2y) - y = 7
  3. Solve the resulting single-variable equation:
    • 24 - 6y - y = 7
    • 24 - 7y = 7
    • -7y = 7 - 24
    • -7y = -17
    • y = 17 7
  4. Substitute y back into the expression for x:
    • x = 8 - 2y
    • x = 8 - 2( 17 7 )
    • x = 8 - 34 7
    • x = 56 7 - 34 7
    • x = 22 7
  5. Check the solution in both original equations: Substitute x = 22 7 and y = s 17 7 into the original equations to verify. For x + 2y = 8:
    • 22 7 + 2 ( 17 7 ) 56 7 = 8

    For 3x - y = 7:

    • 3( 22 7 ) - 17 7 49 7 = 7

  6. Solution: The final result is x = 22 7 , y = 17 7

Elimination method for Solving Systems of Equations

The elimination method involves adding or subtracting the equations in a system to eliminate one of the variables. This reduces the system to a single equation with one unknown, which can be solved directly.
Example:

  • 2x + 3y = 13
    4x - y = 5
  1. Align the equations:
    • 2x + 3y = 13
      4x - y = 5
  2. Make the coefficients of y opposites: Multiply the second equation by 3.
    • 2x + 3y = 13
      3(4x - y) = 3(5)
      12x - 3y = 15
  3. Add the equations to eliminate y:
    • (2x + 3y) + (12x - 3y) = 13 + 15
      14x = 28
      x = 2
  4. Substitute x = 2 back into one of the original equations: Use the first equation.
    • 2x + 3y = 13
      2(2) + 3y = 13
      4 + 3y = 13
      3y = 13- 4
      3y = 9
      y = 3
  5. Check the solution in both original equations:

    For 2x + 3y = 13:

    • 2(2) + 3(3) = 4 + 9 = 13

    For 4x - y = 5:

    • 4(2) - 3 = 8 - 3 = 5
  6. Solution: The final result is x = 2, y = 3.
