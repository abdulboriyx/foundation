Expressions

What are algberaic expressions?

An algebraic expression is a mathematical phrase that includes numbers, variables, and operations (such as addition, subtraction, multiplication, and division). These expressions can represent various quantities and relationships in algebra, but they do not include an equality sign. For example, let's say you want to talk about the number of apples you have. Instead of using a specific number like 5 or 10, you might use a variable like X and say "I have X apples." Here, X can stand for any number of apples.

Types of expressions:

Monomial: An algebraic expression with one term. For example 7x, 3, 4a2

Binomial: An algebraic expression with two terms. For example 3x + 4, a2 - b2

Trinomial: An algebraic expression with three terms. For example x2 + 2x + 1, a3 - 3a + 2

Arithmetic operations with algebraic expressions

Quizzes:

1. If you are buying x apples and y bananas where apples cost $1.5 each and bananas cost $0.75 each, write an algebraic expression to represent the total cost. How would you simplify this expression if you bought 4 apples and 6 bananas?

2. Suppose you save $10 each week, and your friend saves $15 each week. Write an algebraic expression to represent the total amount saved by both of you after w weeks. How much will you both have saved after 8 weeks?

3. If you travel at a constant speed of s miles per hour for t hours, write an algebraic expression for the total distance traveled. How far would you travel if you drive for 5 hours at 60 miles per hour?

4. A recipe calls for 2x cups of flour and x + 3 cups of sugar. Write an algebraic expression for the total number of cups of flour and sugar combined. If x is 2, how many total cups are used?

5. A cell phone plan costs $20 per month plus $0.10 per text message. Write an algebraic expression to represent the monthly cost if you send t text messages. How much would the cost be if you send 50 text messages in a month?
